# AGENTS.md

Project: Numerical Analysis Bilingual Study Website

## Stack
- Next.js App Router
- TypeScript
- Nextra
- MDX
- shadcn/ui

## Content rules
- Every week is one MDX file
- English first, Chinese immediately below
- Use consistent terminology
- Use formal mathematical notation
- Preserve complete derivations and examples
- Use reusable blocks for theorem, example, warning, summary, and practice

## UI rules
- Clean academic style
- Responsive
- Sidebar + TOC
- Good typography for formulas and bilingual text
